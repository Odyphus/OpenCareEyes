# OpenCareEyes
OpenCareEyes是一个开源护眼软件，可以： 

- 🔵 **蓝光过滤** - 通过 GPU Gamma Ramp 调节色温（1000K-6500K），减少蓝光伤害
  
- 🌙 **屏幕调光** - 软件级亮度控制，支持点击穿透的透明遮罩
  
- ⏰ **休息提醒** - 支持番茄钟和 20-20-20 规则，可选强制休息模式
  
- 🎯 **专注模式** - 自动调暗背景窗口，帮助你专注当前任务
  
- 🖥️ **多显示器支持** - 自动适配所有连接的显示器
  
- 🌅 **日出日落调度** - 根据地理位置自动调节色温
  
- ⌨️ **全局热键** - 快捷键快速切换功能
  
- 🚀 **开机自启** - 通过注册表实现开机启动
  
- 🎨 **主题适配** - 自动适配系统亮色/暗色主题

完全免费开源，完美替代付费软件如CareUEyes

源代码在master分支内，软件为release部分的.exe文件，下载后双击即可打开，打开后软件会运行在右下角作为小图标

## 使用方法

1. **启动软件**：双击 `OpenCareEyes_v0.1.1.exe`
2. **查看托盘图标**：软件会在系统托盘（右下角）显示图标
3. **右键菜单**：右键点击托盘图标，可以快速访问所有功能
4. **调节色温**：选择预设色温或自定义数值
5. **设置提醒**：配置休息提醒间隔和时长
6. **退出软件**：右键托盘图标选择"退出"

## 常见问题

**Q: 软件会影响游戏或视频观看吗？**
A: 可以随时通过托盘菜单或快捷键暂时关闭功能。

**Q: 色温调节和系统夜间模式冲突吗？**
A: 不冲突，可以同时使用，效果会叠加。

**Q: 如何卸载？**
A: 直接删除 exe 文件即可，软件不会在系统中留下残留文件（除非启用了开机自启动，需要手动关闭）。
