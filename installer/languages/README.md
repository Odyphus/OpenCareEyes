# Inno Setup language files

`ChineseSimplified.isl` is the official Simplified Chinese translation from
[`jrsoftware/issrc`](https://github.com/jrsoftware/issrc), pinned to commit
`683ee7eabfbce807f901c5da83fc5ff1a3ecb693` for reproducible installer builds.

The translation is used only while compiling the Windows installer. Its
upstream license is preserved in `INNO_SETUP_LICENSE.txt`.
